Privasi Lokal

Data keuangan tetap di browser user

Aplikasi ini tidak mengirim nominal aset, utang, atau catatan ke server. Kalau dipakai di perangkat bersama, aktifkan mode privat, backup ke file JSON lokal, lalu hapus data saat selesai.

  • Tidak ada akun, login, atau sinkronisasi cloud.
  • Backup dan restore berjalan lewat file JSON lokal yang user pilih sendiri.
  • Tombol hapus data bisa dipakai untuk membersihkan browser setelah dipakai.

Prioritas

Urutan utang yang perlu diperhatikan

Ini heuristik sederhana. Umumnya utang berbiaya tinggi, berisiko cepat, atau mengganggu cashflow layak dicek lebih dulu.

Saat belum ada utang aktif, daftar prioritas belum perlu ditampilkan.

Catatan

  • Isi nominal pakai angka saja, misalnya 25000000.
  • Template profil hanya membantu fokus input. Angka aset dan utangmu tidak diubah oleh pilihan profil.
  • Tanggal snapshot, target net worth, dan dana darurat bersifat opsional, tapi membantu membaca progres dengan lebih jernih.
  • Kolom catatan bisa dipakai untuk detail seperti tipe aset, jumlah gram emas, atau nama bank.
  • Data tersimpan otomatis di browser, jadi saat refresh halaman nilainya tetap ada.
  • Kartu komposisi dan prioritas utang di kanan hanya panduan awal, bukan pengganti keputusan finansial pribadi.
  • Proyeksi bukan jaminan hasil masa depan. Gunakan sebagai simulasi kasar, bukan angka pasti.
  • Simpan snapshot saat ada perubahan besar, misalnya setelah gajian, pelunasan utang, atau update nilai aset.
  • Yang sering lupa dihitung: giro, simpanan koperasi, RSU vested, stok usaha, DPLK, bonus tertagih, utang vendor, dan tagihan medis.

🔒 Keamanan & Privasi Data

✅ Data Aman di Perangkat Anda

Semua informasi keuangan disimpan hanya di browser lokal perangkat Anda. Data tidak pernah dikirim ke server atau pihak ketiga manapun.

🔐 Fitur Keamanan

  • Tidak ada koneksi internet — aplikasi offline
  • Protection headers lengkap (CSP, X-Frame-Options, dll)
  • Mode privat tersedia (hapus otomatis saat browser ditutup)
  • Tidak ada script eksternal atau tracking

⚠️ Penting untuk Diperhatikan

  • Jaga keamanan perangkat — update browser dan hindari malware
  • Shared device? — gunakan Mode Privat agar data hilang saat browser ditutup
  • Tanpa enkripsi — data disimpan sebagai plaintext di localStorage

💡 Saran: Hanya gunakan untuk tracking dan perencanaan. JANGAN simpan password atau token sensitif di sini.
